# **App Name**: SPI Hub

## Core Features:

- Centralized Operational Calendar: A unified interface aggregating staff movements and leave requests with role-based privacy filters to see teammate availability.
- Role-Based Staff Directory: Management-only interface to create and manage employee profiles, departments, and specific annual leave or claim limits.
- Dynamic Movement Logger: Structured logging system for site visits and meetings including GPS-tagged destination details, project association, and mileage claim integration.
- Leave & Claims Workflow: Full lifecycle tracking for leave applications and medical/general claims with manual file verification for receipts.
- Project-Linked Ticket System: Issue tracking tied to specific active projects, allowing HODs to assign technical tasks and monitor severity levels and resolution status.
- AI Ticket Intelligence Tool: A feature for Project Leads to generate an automated executive summary tool from long ticket logs to prioritize department workflows efficiently.
- Secure Relational Storage: SQL-based database management (PostgreSQL/MySQL) ensuring referential integrity across staff records, movement logs, and claim histories.

## Style Guidelines:

- Primary Color: Professional Indigo (#1A42F0) representing technical precision and authority.
- Background: Deep Charcoal (#111216), providing a high-end dark workspace appropriate for administrative focus.
- Accent: Bright Cyan (#3FD5F1) to highlight active tickets and navigation call-to-actions.
- Headline Font: 'Space Grotesk' for a technical, scientific, and machined feel in headers and dashboards.
- Body Font: 'Inter' for clean readability across data-heavy forms, logs, and calendar tooltips.
- Monochromatic line icons to denote departments, status types, and movement categories (Logistics vs Client Meetings).
- Single-tenant internal dashboard layout featuring a persistent left navigation sidebar and high-density data tables.
